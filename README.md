## Pour commencer
Clonez le dépôt.
```
bundle install
```
```
rails db:create
```
```
rails db:migrate
```
```
rails db:seed
```

Si on veut accéder au site en local :
```
rails server
```
Pour accéder au site : http://localhost:3000/

## Prérequis
Ruby 2.5.1
Rails 5.2.1
